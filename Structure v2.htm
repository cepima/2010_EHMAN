BASIC STRUCTURE OF PROCEL-SC

Middleware is computer software that connects software components or some people and their applications. The software consists of a set of services that allows multiple processes running on one or more machines to interact.

THOT is an intelligent platform for real time management of decision making at different levels of the production scenario. THOT acts as a repository of all information that is available to it, and it feeds such information to the Decision Maker Model (DMM). Each OPC is a stand alone application that provides with different variables (input/output) depending on where it is connected to. Its structure is fixed and has to be coded for each application. The specification defined a standard set of objects, interfaces and methods for use in process control and manufacturing automation applications to facilitate interoperability.

The Decision making model (DMM) acts controlling the actual information flows connecting the different Supply Chain echelons. The DMM Is in charge of deciding what is the SC going to do.

Decisions considered by this DMM could encompass, for example in order to follow a given inventory policy the DMM, is in charge of one of more of the following actions:

·         Opening and closing valves that connect different plants.

·         Setting tanks set points.

·         Fixing flow rates to different values.
